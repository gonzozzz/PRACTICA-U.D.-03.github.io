¿Qué es GitHub Pages?

GitHub Pages es un servicio que permite alojar páginas web estáticas directamente desde un repositorio de GitHub.

Esto significa que puedes publicar una web sin necesidad de servidor propio, ya que GitHub se encarga de mostrar los archivos HTML, CSS y JavaScript en Internet.

Este servicio se utiliza habitualmente para crear documentación, portafolios o páginas de proyectos. :contentReference[oaicite:0]{index=0}

1. Crear un repositorio

El primer paso consiste en crear un repositorio en GitHub. Este repositorio contendrá todos los archivos de la página web.

GitHub Pages permite usar tanto repositorios nuevos como existentes para publicar un sitio web. :contentReference[oaicite:1]{index=1}

El nombre del repositorio puede ser cualquiera si se trata de un proyecto, aunque también existe la opción de usar un nombre especial para páginas personales.

2. Subir los archivos de la web

Una vez creado el repositorio, es necesario subir los archivos de la página web. Estos pueden ser archivos HTML, CSS y JavaScript.

GitHub Pages buscará automáticamente un archivo principal llamado:

index.html

También puede usar index.md o README.md, que serán procesados como contenido web. :contentReference[oaicite:2]{index=2}

3. Configurar la publicación

Para que la página sea visible en Internet, es necesario configurar la fuente de publicación.

GitHub permite elegir una rama (por ejemplo main) y una carpeta desde donde se publicará la web. :contentReference[oaicite:3]{index=3}

  1. Ir a Settings del repositorio
  2. Entrar en la sección Pages
  3. Seleccionar la rama de publicación
  4. Guardar los cambios

4. Publicación y acceso

Una vez configurado, GitHub genera automáticamente el sitio web y lo publica en Internet.

La dirección de la página dependerá del tipo de repositorio:

https://usuario.github.io/repositorio

La publicación puede tardar unos minutos en completarse después de subir los cambios. :contentReference[oaicite:4]{index=4}

Cada vez que se actualicen los archivos del repositorio, la página web se actualizará automáticamente.
